# Medical Appointments API

## Description
REST API for managing medical appointments, developed using Scrum.

## Tech Stack
- Java
- Spring Boot
- PostgreSQL
- JPA / Hibernate
- JWT (planned)

## Architecture
- Layered architecture (Controller, Service, Repository)
- SQL for transactional data
- RESTful API


## How to Run
1. Clone repository
2. Configure application.properties
3. Run Spring Boot application
